*crear carpetas en git.
mkdir "repo_name"
*remover carpetas en git
rmdir "repo_name."
*inicializar git
git nit
*mostrar directorios
ls
*mostrar directorios ocultos
ls -a
*git add "name_archivo"
subir archivo a la area de preparacion, o se està listo para subir al repositorio
*git status
mostrar informaciòn del directorio de trabajo y el area de preparacin
*git rm --cached "name_file". -->

*git commit  -m "nombre que quieras, es decir una descripcion" -a , el "-m" es para el mensaje, el -a es para agregar al estado stagubd,
sube todo los archivos  al repositorio.
